重拾博客一年多

By 三疯

发表于 2013-11-05

文章目录

博客断断续续写了好几年，重拾博客的第一篇是 2012 年 9 月 19 日写的吐槽 AS3 的接口。这一年多每月都坚持写个两三篇，纪念下。

这些年在多个地方写过（我的技术博客初衷）， 1 年前的决定定在 C++ 博客园了。最近又看到些用 Github Pages 写博客的文章，有股搞一搞的暗劲。

在 GitHub 写博客的优势就是我一直想要的一个功能，写的文章有版本管理，当自己修改某些文章的时候，看下历史记录就知道修改的原因了。另外隐约感觉 GitHub 应该比博客园活的久。

但是放 GitHub 之后，会有人看吗，搜索引擎能很快收录吗？不得不再次思考自己博客的初衷……

慢慢去想吧，无论放到哪，一定要坚持更新……

这一年多也有不少心得和经验没有写到博客里，有的可能是忙完后忘了总结，有时根本都没意识到那个东西挺值得总结的。另外这一年的博客也有些有充数嫌疑，汗，这点慢慢改正。

特别讨厌很多博客转载别人的文章，所以我的博客里肯定是坚持原创，至少也是一番搜索了解某项技术后的总结（并给出原出处的链接且不赘述人家写完的）。

一直在考虑要不要写点技术外的东西，例如硬件相关的。想了下，硬件也是程序员的工具（工欲善其事必先利其器）啊，工具又不一定是软件写成的 tool，所以以后碰到也可以写点儿。
